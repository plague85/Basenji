Basenji
=======

:Autor: jRiddick
:Release: |release|
:Date: |today|

**Basenji** is a fast, scalable and awesome Usenet indexer. It only exposes an RESTful API
so you can use it wherever you want. 

Planned Features
----------------

 * RESTful API using Twisted or Tornado
 * Fast header downloading using multiple processes
 * Metadata from IMDb, TMDb, RottenTomatoes, TVRage and more
 * Make releases with metadata information from the headers
 * ElasticSearch for searching the data
 * NZB creation from a JSON document
